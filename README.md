# Turbial Components

Monorepo for Turbial's UI component tools.

## Packages

### `packages/web-components`
Extracts brand colors, fonts, images, and structure from any webpage.

```bash
npx github:Turbial/components --source <url> --out data.json
```

### `packages/component-fetch`
Pulls UI components from shadcn, magicui, 21st.dev, hyperui.

```bash
npx github:Turbial/component-fetch --source shadcn --name button --out ./out
```
